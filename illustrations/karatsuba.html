    Pseudo-code (Karatsuba)
    function karatsuba(x, y):
    n = max(len(x), len(y))
    if n == 1: return x*y
    m = floor(n/2)
    xL, xR = split(x, m)
    yL, yR = split(y, m)
    z0 = karatsuba(xR, yR)
    z2 = karatsuba(xL, yL)
    z1 = karatsuba(xL+xR, yL+yR) - z0 - z2
    return (z2 << 2*m) + (z1 << m) + z0
